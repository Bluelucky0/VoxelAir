# VoxelAir
项目介绍：模拟官方网站的PC端应用.
实现功能：每一屏页面的搭建与交互，实现开机动画，鼠标滚动，3D轮播，机器人动画，还有canvas气泡动画效果等.
技术点：DOM中位置尺寸API的使用，用CSS3实现机器人的动画效果，用Canvas画布实现气泡效果等.
